##Flujo de navegacion 

El usuario ingresa al sistema y se registra o inicia sesión. Luego selecciona una especialidad o médico disponible, elige fecha y horario, confirma el turno y recibe una notificación de confirmación.

Recorrido:

Inicio → Inicio de sesión → Selección de médico → Seleccion de fecha → Confirmación → Notificación
